# SweetWhisk Pastel Delights

A modern bakery website built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Getting Started

Prerequisites: Node.js (LTS) and npm installed.

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui
